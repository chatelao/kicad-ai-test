# NE555 Timer Blinking LED Circuit

## Requirements
- Power Supply: 5V - 12V DC.
- Frequency: ~1Hz (1 blink per second).
- Components:
  - NE555P (DIP-8 package)
  - R1: 1k
  - R2: 470k
  - C1: 1uF
  - LED (5mm)
  - Current limiting resistor for LED: 220 Ohm
  - Decoupling capacitor: 10nF (optional but recommended)

## 555 Pinout (DIP-8)
1. GND
2. TRIGGER
3. OUTPUT
4. RESET (Active Low, connect to VCC)
5. CONTROL VOLTAGE
6. THRESHOLD
7. DISCHARGE
8. VCC
