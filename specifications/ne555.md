# NE555 Precision Timer Datasheet (Summary)

The NE555 is a precision timing circuit capable of producing accurate time delays or oscillation.

## Pin Configuration (SOIC-8 / DIP-8)
1. **GND**: Ground (0V)
2. **TRIG**: Trigger input (1/3 Vcc)
3. **OUT**: Output (high/low)
4. **RESET**: Reset input (active low)
5. **CONT**: Control voltage (2/3 Vcc)
6. **THRES**: Threshold input (2/3 Vcc)
7. **DISCH**: Discharge (open collector)
8. **Vcc**: Supply voltage (typically 4.5V to 15V)

## Absolute Maximum Ratings
- Supply Voltage (Vcc): 18V
- Output Current: 200 mA
- Operating Free-Air Temperature: 0°C to 70°C

## Astable Operation (Blinking)
- Frequency $f = \frac{1.44}{(R1 + 2R2)C}$
- Duty Cycle $D = \frac{R1 + R2}{R1 + 2R2}$

For a ~1Hz blink with a 1uF capacitor:
- $R1 = 10k\Omega$
- $R2 = 680k\Omega$
- $C = 1uF$
- $Vcc = 5V$
- $R_{LED} = 330\Omega$ for Red LED.
