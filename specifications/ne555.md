# NE555 Precision Timers

## General Description
The NE555 is a precision timing circuit capable of producing accurate time delays or oscillation. In the time-delay or monostable mode of operation, the timed interval is controlled by a single external resistor and capacitor network. In the astable mode of operation, the frequency and duty cycle are controlled independently with two external resistors and one external capacitor.

## Pin Configuration
1. GND: Ground
2. TRIG: Trigger
3. OUT: Output
4. RESET: Reset
5. CONT: Control Voltage
6. THRES: Threshold
7. DISCH: Discharge
8. VCC: Supply Voltage

## Features
- Timing from microseconds through hours
- Operates in both astable and monostable modes
- Adjustable duty cycle
- Output can source or sink 200 mA
- TTL compatible output
- Temperature stability better than 0.005% per °C

## Astable Operation
In the astable mode, the 555 timer puts out a continuous stream of rectangular pulses having a specified frequency.
- f = 1.44 / ((RA + 2RB) * C)
- Duty Cycle = (RA + RB) / (RA + 2RB)
