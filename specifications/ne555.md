# NE555 Timer Specifications

## Pins
1. GND
2. TRIGGER
3. OUTPUT
4. RESET
5. CONTROL VOLTAGE
6. THRESHOLD
7. DISCHARGE
8. VCC

## Astable Multivibrator Configuration
Components:
- R1 (between VCC and DISCHARGE)
- R2 (between DISCHARGE and THRESHOLD/TRIGGER)
- C1 (between THRESHOLD/TRIGGER and GND)
- LED (connected to OUTPUT through resistor)
- C2 (optional, 10nF between CONTROL and GND)
