# NE555 Datasheet Technical Summary

## General Description
The NE555 is a precision timing circuit capable of producing accurate time delays or oscillation. In the time-delay or mono-stable mode of operation, the timed interval is controlled by a single external resistor and capacitor network. In the a-stable mode of operation, the frequency and duty cycle are controlled independently with two external resistors and one external capacitor.

## Pin Configuration
1. **GND**: Ground (0V)
2. **TRIG**: Trigger (starts the timing cycle, active low)
3. **OUT**: Output
4. **RESET**: Reset (active low)
5. **CONT**: Control Voltage (allows access to the internal voltage divider)
6. **THRES**: Threshold (ends the timing cycle)
7. **DISCH**: Discharge (used to discharge the timing capacitor)
8. **VCC**: Supply Voltage (+4.5V to +16V)

## Astable Operation (Blinking LED)
- **Frequency (f)**: $f = \frac{1.44}{(R_1 + 2R_2)C}$
- **High Time ($t_H$)**: $t_H = 0.693(R_1 + R_2)C$
- **Low Time ($t_L$)**: $t_L = 0.693 R_2 C$
- **Period (T)**: $T = t_H + t_L = 0.693(R_1 + 2R_2)C$
- **Duty Cycle**: $D = \frac{R_1 + R_2}{R_1 + 2R_2}$

## Design for 1Hz, 30% Active (Low), 70% Dark (High)
*Note: Usually "active" means the LED is ON. If the LED is connected from VCC to OUT, it's ON when OUT is LOW.*
If we want 30% active (LED ON) and 70% dark (LED OFF):
- If LED is connected to GND: $t_H = 0.3s$, $t_L = 0.7s$ (Not possible with standard 555 circuit as $t_H > t_L$)
- If LED is connected to VCC: $t_L = 0.3s$, $t_H = 0.7s$ (This works!)

$t_L = 0.693 R_2 C = 0.3s$
$t_H = 0.693 (R_1 + R_2) C = 0.7s$

$\frac{t_H}{t_L} = \frac{R_1 + R_2}{R_2} = \frac{0.7}{0.3} = 2.33$
$1 + \frac{R_1}{R_2} = 2.33 \Rightarrow \frac{R_1}{R_2} = 1.33 \Rightarrow R_1 = 1.33 R_2$

Pick $C = 10\mu F$
$0.3 = 0.693 \cdot R_2 \cdot 10 \cdot 10^{-6}$
$R_2 = \frac{0.3}{0.693 \cdot 10^{-5}} = \frac{30000}{0.693} \approx 43290 \Omega = 43.3 k\Omega$
$R_1 = 1.33 \cdot 43.3 \approx 57.6 k\Omega$

Using standard E24 values:
$R_2 = 43 k\Omega$
$R_1 = 56 k\Omega$

Check:
$t_L = 0.693 \cdot 43k \cdot 10u = 0.298 s$
$t_H = 0.693 \cdot (56k + 43k) \cdot 10u = 0.693 \cdot 99k \cdot 10u = 0.686 s$
$T = 0.298 + 0.686 = 0.984 s \approx 1Hz$
Duty cycle (High) = 69.7%
Duty cycle (Low) = 30.3%
