# NE555 Technical Specifications

## Pinout
1. **GND**: Ground (0V)
2. **TRIG**: Trigger (starts the timing cycle)
3. **OUT**: Output (provides the timed pulse)
4. **RESET**: Reset (interrupts the timing cycle)
5. **CTRL**: Control Voltage (adjusts the threshold voltage)
6. **THRES**: Threshold (ends the timing cycle)
7. **DISCH**: Discharge (discharges the timing capacitor)
8. **VCC**: Supply Voltage (+Vcc)

## Operating Parameters
- **Supply Voltage (Vcc)**: 4.5V to 16V
- **Supply Current (Vcc = 5V)**: 3mA to 6mA
- **Supply Current (Vcc = 15V)**: 10mA to 15mA
- **Max Output Current**: 200mA
- **Power Dissipation**: 600mW

## Astable Mode Configuration
In astable mode, the NE555 operates as an oscillator. The output is a square wave.

### Components
- **R1**: Resistor between Vcc and Discharge pin.
- **R2**: Resistor between Discharge pin and Threshold/Trigger pins.
- **C**: Capacitor between Threshold/Trigger pins and Ground.

### Timing Formula
The frequency $f$ and duty cycle $D$ are given by:
- $t_{high} = 0.693 \times (R1 + R2) \times C$
- $t_{low} = 0.693 \times R2 \times C$
- $T = t_{high} + t_{low} = 0.693 \times (R1 + 2 \times R2) \times C$
- $f = \frac{1}{T} = \frac{1.44}{(R1 + 2 \times R2) \times C}$
- $D = \frac{t_{high}}{T} = \frac{R1 + R2}{R1 + 2 \times R2}$
