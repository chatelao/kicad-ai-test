# 555 Timer IC Specifications

## Overview
The 555 timer IC is an integrated circuit used in a variety of timer, delay, pulse generation, and oscillator applications.

## Pinout (DIP-8)
1. GND - Ground
2. TRIGGER - Start timing interval when voltage falls below 1/3 Vcc
3. OUTPUT - Push-pull output
4. RESET - Timing reset (active low)
5. CONTROL - Access to internal voltage divider (2/3 Vcc)
6. THRESHOLD - End timing interval when voltage is above 2/3 Vcc
7. DISCHARGE - Open-collector output to discharge capacitor
8. Vcc - Positive supply (4.5V to 16V for NE555)

## Astable Mode (Oscillator)
- Frequency f = 1 / (ln(2) * (R1 + 2*R2) * C)
- Duty Cycle D = (R1 + R2) / (R1 + 2*R2)
- High Time t_high = ln(2) * (R1 + R2) * C
- Low Time t_low = ln(2) * R2 * C
