# 555 Timer Blinking LED Specification

## Components
- **U1**: NE555 Precision Timer (8-pin DIP)
- **R1**: 1k Ohm resistor
- **R2**: 100k Ohm resistor
- **R3**: 470 Ohm resistor (LED current limiter)
- **C1**: 10uF electrolytic capacitor (timing)
- **C2**: 0.01uF ceramic capacitor (control voltage decoupling)
- **D1**: Red LED
- **VCC**: 5V - 15V DC
- **GND**: 0V

## Circuit Configuration (Astable Multivibrator)
1. **GND (Pin 1)** to Ground
2. **Trigger (Pin 2)** to Threshold (Pin 6)
3. **Output (Pin 3)** to R3 which connects to LED anode. LED cathode to Ground.
4. **Reset (Pin 4)** to VCC
5. **Control Voltage (Pin 5)** to Ground via C2 (0.01uF)
6. **Threshold (Pin 6)** to Trigger (Pin 2) and to R2-C1 junction
7. **Discharge (Pin 7)** to R1-R2 junction
8. **VCC (Pin 8)** to VCC

## Timing Formulas
- **High Time (t1)**: 0.693 * (R1 + R2) * C1
- **Low Time (t2)**: 0.693 * R2 * C1
- **Period (T)**: t1 + t2 = 0.693 * (R1 + 2*R2) * C1
- **Frequency (f)**: 1 / T = 1.44 / ((R1 + 2*R2) * C1)
